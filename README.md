<h1>AI X-rayScan</h1>

<h3>Overview</h3>
<p>This is a web-based system designed to assist users in identifying possible diseases based on provided X-ray images. The platform provides an intuitive interface where users can input their X-rays and receive predicted health conditions along with relevant information.

This project aims to demonstrate the practical application of web development concepts in healthcare-related problem solving.</p>

<h3>How It Works</h3>
<ul>
  <li>The user uploads an X-ray image through the web interface</li>
  <li>The image is sent to a Python backend via an HTTP POST request</li>
  <li>The backend preprocesses the image (resize, grayscale, normalization)</li>
  <li>A trained CNN model analyzes the X-ray</li>
  <li>The model predicts the most likely disease class</li>
  <li>The result is returned as JSON and displayed in the browser</li>
  <p>The system runs locally and uses a FastAPI backend connected to a browser-based frontend for real-time predictions.</p>
</ul>
